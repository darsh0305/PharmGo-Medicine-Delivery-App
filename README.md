## 📱 Overview

**PharmGo** is an Android medicine delivery application developed using **Java** and **XML**. The application provides an easy and efficient way for users to browse medicines, search for products, manage their shopping cart, and place orders through an intuitive interface.

---

## ✨ Features

- 🔐 User Authentication
- 💊 Browse Medicines
- 🔍 Search Medicines
- 🛒 Shopping Cart
- 📂 Medicine Categories
- 📱 Clean and Responsive UI
- 🔥 Firebase Integration
- 📦 Order Management
- 👤 User Profile
